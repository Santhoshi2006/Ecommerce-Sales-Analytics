# 📊 Ecommerce Sales Analysis Dashboard

## 🔍 Objective
Analyze e-commerce sales data to identify trends, top products, and customer behavior.

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📈 Dashboard Features
- Revenue Trends
- Top Selling Products
- Country-wise Sales
- Monthly Analysis
- Top Customers

## 💡 Key Insights
- Top products generate major revenue
- UK dominates sales
- Seasonal trends observed
- Few customers contribute high revenue

## 🚀 Recommendations
- Focus on top products
- Expand in top countries
- Improve low sales months
- Retain loyal customers
